Purpose-built for extensibility.

The toolkit centers on a clean command registry, typed services, and isolated adapters so every new capability is a predictable add-on. You can register new commands in minutes without touching existing logic.

It ships with web research, Gemini feedback, and image generation commands. Each one saves outputs in a consistent folder structure so work can be audited and reused.

What is AI orchestration?

AI orchestration is directing an AI system through a workflow of goals, tools, and constraints so outputs are reliable and repeatable. You are managing a workforce of experts, not just asking questions.

Think like a CEO: set the strategy, define quality standards, and control the context. Then think like a sommelier and barista: select the right model and make the process consistent.

Automation mindset: any time you repeat a task (searching, screenshotting, generating images), design a tool and a process that removes you from the loop.

What You Will Build

Each command stores its output in a predictable location.

Web Search

Research online with OpenAI web search tools.

npm run dev -- web-search "Gemini 3 API upload"
Output: references/

Gemini Response

Send prompts or media to Gemini and archive responses.

npm run dev -- gemini "Summarize this image" --file ./img.png
Output: references/aI_feedback/

Image Generate

Create images from prompts with prompt-based filenames.

npm run dev -- image-generate "Cinematic lighthouse"
Output: images/
